## 三、数据质量

### 3.1 标题质量

- [x] 标题无日期前缀残留（如 "04-07"、"2026-05-19"）
- [x] 标题无日期后缀残留
- [x] 标题无 HTML 实体或标签
- [x] 标题长度合理（>2 字符）

### 3.2 日期格式

- [x] 统一为 `YYYY-MM-DD`（如 2026-05-19）
- [x] 或 `MM-DD`（如 05-19，仅当年）
- [x] 无 `[2026-05-19]` 括号格式
- [x] 无 `2026/05/19` 斜杠格式
- [x] 无 `2026年5月19日` 中文格式
- [x] 无 `5月13日` 混合格式

### 3.3 URL 有效性

- [x] URL 以 https:// 开头
- [x] URL 包含 `/info/` 路径段
- [x] URL 可在浏览器中打开
